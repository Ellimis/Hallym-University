퀴즈 정답을 제출하실 때는 py_06_퀴즈_학번_이름.txt 에 저장하여 첨부하시면 됩니다.

이렇게 하는 이유는, 우리가 컴파일러를 사용하여 프로그램을 작성하게 되는데, 퀴즈 관리에서는 그렇게 작성된 소스를 복사 붙여 넣기를 할 수 없기 때문입니다. 여러분들께서 일일이 타이핑하는 수고를 덜어드리려고 과제관리에 제출하도록 하는 것이니 첨부파일명을 규칙에 맞게 만들어 주시기 바랍니다.

[06주차_퀴즈_문제] py_06주차_퀴즈.txt

0을 입력하기 전까지 숫자를 입력받아 data 리스트에 저장합니다. 
data 리스트에 저장된 값들이 오르락내리락 하며 변화 하고 있을 때, 
그 변화 과정에서 고점을 찍은 index 만을 peak_index 리스트에 저장하려고 합니다. 
단, 마지막 값과 첫 값은 연속된 과정이라 생각합니다. 예를 들어,

data =  [8, 7, 6, 9, 4, 3, 5] 이라면 peak_index = [0, 3]
data =  [4, 7, 6, 9, 1, 6, 5] 이라면 peak_index = [1, 3, 5]
data =  [4, 7, 6, 9, 1, 6, 8] 이라면 peak_index = [1, 3, 6]

0을 입력하기 전까지 숫자를 입력받아 data 리스트에 저장하고 peak_index 리스트를 출력하세요.
